Hudson kann Debt Reports einlesen und darstellen. Wenn dies aktiviert wird, analysiert Hudson nach jedem Build die erzeugten Debt Dateien und stellt deren Informationen in verschiedenen Ansichten dar: Trend Anzeige, Projekt Statistik und Package Statistik. Zudem zeigt Hudson die jeweiligen Warnungen direkt in der betroffen Datei an.
